### Windows 常用命令
```Plain
> where ipconfig                        # 查看命令所在位置
> prompt $P$G$T$G                       # Windows CMD 显示时间
> findstr /s /i /n /c:"search_str" *.*  # Window 全文搜索
```

### Windows 程序
```Plain
> calc                                  # 计算器
> regedit                               # 注册表编辑器
> notepad                               # 记事本
> msconfig                              # 系统配置
> gpedit                                # 组策略编辑器
> snippingtool                          # 截图工具
> mstsc                                 # 远程桌面
```
