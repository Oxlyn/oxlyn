5. 标识符及关键字

  1. 标识符

    • 标识符由数字、字母、下划线和$组成，不能以数字开头
    • 大小写敏感
    • 不能使用java中的关键字和保留字
    • 包名所有字母小写
    • 类名每个单词首字母大写
    • 变量和方法：第一个单词小写，其他单词首字母大写
    • 常量所有字母大写，单词之间用“_”连接。

  2. 常用的转义字符

    • \b 退格
    • \f 换页
    • \n 回车
    • \r 换行
    • \t 制表符Tab
    • \’ 单引号
    • \” 双引号
    • \ 反斜杠
